◆ parseRect()

QRectF dsqt::parseRect ( const std::string & s)

Parses a string into a rectangle.

Example: size="400, 400, 0, 0", where it's "L, T, W, H" the space after the comma is required to read the second and third token. Defaults parameters to 0 if they don't exist.
